## Details
The entire core interaction loop has been overhauled from a rudimentary "angry birds drag" to a tactical, glassmorphism 3D heads-up display. We also completely swapped out the loader for a high-intensity 3D cinematic. 

### Changes Made

**1. The Preloader Cinematic**
- `Preloader.tsx` was replaced by `IntroCinematic.tsx` — a native React Three Fiber component that renders a high-speed chase sequence with aggressive camera tracking. It yields out seamlessly back into the game state without forcing a hard refresh.

**2. The Predictive Slingshot**
- **Event Triggers:** Reverted back to the fluid, tactile click-and-drag motion. 
- **The Glow Line:** Implemented `TrajectoryLine.tsx` natively on the Rapier game loop. A **glowing neon dotted line** mathematically simulates the precise parabolic arc to project the EXACT path the ball will take.
- **Dynamic Physics:** The line dynamically adjusts based on the active physics environment (zero-gravity vs. planet surface).
